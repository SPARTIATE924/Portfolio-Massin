La Josslan

Projet Formation - LAN

Description du projet

La Josslan est un projet réalisé dans le cadre de ma formation en SISR. Nous y avons monté un projet de LAN afin de pouvoir jouer ensemble sur un réseau local.

Pour ce faire, nous avons installé :

  • 10 postes de travail
  • Switches et Routeurs
  • Serveur installé sur ESXi avec Windows Server 2019
  • Service DHCP configuré pour le projet
  • Serveur de jeu CounterStrike

Une fois tout le matériel configuré, nous avons pu jouer ensemble en local sur le jeu CounterStrike.
